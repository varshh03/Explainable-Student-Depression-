# Explainable-Student-Depression-
Student Depression Detection System built using Machine Learning and Explainable AI (XAI). It predicts the likelihood of depression in students and provides insights into the key factors influencing the prediction. The goal is to support early detection, awareness, and better decision-making.


🚀 Features
🔍 Predicts student depression levels
🧾 Provides explainable results (XAI)
📊 Identifies key contributing factors (stress, sleep, academics, etc.)
🌐 User-friendly interface (Streamlit)
⚡ Fast and efficient predictions


🛠️ Tech Stack
Frontend: Streamlit
Backend: Python
Machine Learning: Scikit-learn / TensorFlow
XAI Tools: SHAP / LIME
Data Handling: Pandas, NumPy
